# Stack Buffer Overflow

## Stack Buffer Overflow
- **Buffer** : 데이터가 목적지로 이동되기 전에 보관되는 임시 저장소
  - 빠른 속도로 이동하던 데이터가 안정적으로 목적지에 도달할 수 있도록 완충 작용을 함
- **Buffer Overflow** : 버퍼가 넘치는 것

### 버퍼 오버플로우 공격 예시
- 중요 데이터 변조
  - 버퍼 오버플로우가 발생하는 버퍼 뒤에 중요한 데이터가 있다면, 해당 데이터가 변조됨으로써 문제가 발생할 수 있음
-  데이터 유출
  - C언어에서 정상적인 문자열은 널 바이트로 종결되고, 표준 문자열 출력 함수들은 널 바이트를 문자열의 끝으로 인식함
  - 어떤 버퍼에 오버플로우를 발생시켜 다른 버퍼와의 사이에 있는 널 바이트를 모두 제거하면 다른 버퍼의 데이터도 같이 출력할 수 있음
- 실행 흐름 조작
  - 함수의 반환 주소를 조작하면 프로세스의 실행 흐름을 바꿀 수 있음
