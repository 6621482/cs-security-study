# Netcat
- TCP 또는 UDP를 사용하여 네트워크 연결을 읽거나 기록하는 컴퓨터 네트워킹 유틸리티

## nc 사용법
- `nc [options] <host> <port>` (`nc -l [options] <port>`)
  - 기본적으로 TCP 사용
  - UDP 연결 : `nc -u host port`
  - options
    - `-p` : 포트 번호 지정
    - `-v` : 연결 과정에서 더 자세한 정보를 출력
    - `-w` : 타임아웃 시간을 초 단위로 설정
    - `-l` : 서버모드로 실행하여 들어오는 연결을 기다림

## 예시
### 1. TCP 서버 열기
- Netcat을 서버 모드로 실행하여 특정 포트에서 대기할 수 있음
  ```
  nc -l -p 1234
  ```
- 클라이언트 연결
  ```
  nc 서버_IP 1234
  ```
- 연결이 완료되면 서버와 클라이언트 간에 채팅처럼 데이터 전송이 가능함

### 2. 파일 전송하기
- 서버 (파일 받기)
  ```
  nc -l -p 1234 > received_file.txt
  ```
- 클라이언트 (파일 보내기)
  ```
  nc 서버_IP 1234 < myfile.txt
  ```
- receivec_file.txt에 myfile.txt의 내용이 저장됨
- 참고
  - < (입력) : 파일 -> 네트워크
  - \> (출력) : 네트워크 -> 파일
